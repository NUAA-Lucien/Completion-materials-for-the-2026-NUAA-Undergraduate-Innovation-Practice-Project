# Completion Materials for the 2026 NUAA Undergraduate Innovation Practice Project

This repository contains the completion materials for the 2026 NUAA Undergraduate Innovation Practice Project.

## Contents

- `研究总结报告.docx`: final project summary report
- `结题答辩PPT.pptx`: completion defense presentation
- `创新训练项目展板.docx`: project poster board material
- `成果统计表.xls`: achievement statistics table
- `软件exe版.zip`: packaged executable release
- `abaqus_case_tool/`: source code and related assets for the project tool

## Notes

- `软件exe版.zip` is stored with Git LFS because it exceeds GitHub's normal file size limit.
- If you clone this repository and want the full executable package, make sure Git LFS is installed before pulling files.

## Clone

```bash
git lfs install
git clone https://github.com/NUAA-Lucien/Completion-materials-for-the-2026-NUAA-Undergraduate-Innovation-Practice-Project.git
```
